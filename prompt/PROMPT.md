삼성 헬스 브릿지 앱 개발 할꺼야
https://github.com/danielk0121/healthputout
요구사항은 url 깃허브 리드미에 있어

### 기본 지침
- "[x]" 표시 항목은 이미 구현된 항목이므로 언급하지 않고, 참고만 한다
- "[ ]" 표시 항목은 미구현 항목이므로 구현을 위해 코드를 작성한다
- 미구현 항목 중 가장 상단에 있는 항목을 먼저 구현하되, 한번에 하나의 항목만 코드 작성한다
- 혹시 가장 상단에 있는 미구현 항목을 구현하지 못하면 이유를 알려주고, 할일을 추가할 수 있도록 알려준다

### 목적
- 지금 삼성 헬스에 있는 체중 기록 데이터를 읽어오는 기능만 구현 되었어
- 텍스트 박스에 csv 문법 형태로 날짜시간,체중을 입력하면 삼성 헬스에 데이터를 쓰는 기능을 만들꺼야

### xml 레이아웃 생성
- [x] 탭 2개로 화면 구성
- [x] 조회 탭
  - [x] 내 체중 기록 조회 등 제목으로 보이는 항목 삭제
  - [x] 데이터 불러오기 버튼
  - [x] 데이터 내용 출력 박스
  - [x] 단 2개만 존재
- [x] 기록 탭
  - [x] "데이터 직접 입력" 등 제목으로 보이는 항목 삭제

### 조회 기능 구현
- [x] 버튼 누르면 삼성 헬스 데이터 조회해서 출력하는 기능

### 기록 기능 구현
- [ ] 날짜,체중 데이터를 입력하고 버튼을 누르면 삼성 헬스에 저장하는 기능
  - [ ] 테스트 데이터 1개 입력해서 테스트 하고, 조회 되는지 확인
    - [ ] 테스트를 위해 텍스트 박스에 하드 코딩
    - [ ] 하드 코딩 내용을 주석 제거
